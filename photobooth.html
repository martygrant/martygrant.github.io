Raspberry Pi Wedding Photobooth Hobby Project

Project Description

I created a wedding photobooth both for fun and to try to save some money for a family member's wedding who would otherwise rent one. The photobooth itself is a simple box which we put together and had a CNC workshop to cut some holes for a screen, buttons and the camera. The photobooth operates using a Raspberry Pi + camera module and some arcade style LED buttons. When a user initiates the photobooth, a countdown starts the screen will show a camera preview and then capture a photo when the countdown finishes. They then get an option to start over if they don't like the photo, or to save the photo.

The first version of the photobooth only saves the photo to a file (also an external drive and cloud storage) as there were problems with printing from the Raspberry Pi and I did not have enough time to fix this for the wedding. The photobooth has space for a printer to sit inside with a slot for the printed photos to come out of. I am continuing to work on the photobooth for my own wedding to add printing and some other extra features like creating a GIF during the camera countdown preview. I'm not making the code public at the moment as it needs heavily tidied up but will be available soon.
