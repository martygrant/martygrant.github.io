Advanced Games Programming 1 Coursework

Project Description

For this coursework I created a OpenGL demo to explore lighting. The demo is a simple scene with three controllable lights and blends them together. The properties of each light can be controlled at runtime, letting you change the colour, position and attenuation. It uses OpenGL and GLSL 3.3 and implements the Phong lighting model.
