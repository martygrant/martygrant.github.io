Discord Bot Open Source Project

Project Description

The UWS Game Dev Society started a Discord chat server in 2018 and there was an interest expressed in developing a custom chat bot for the server. I started development of Bjarne, the bot for the server. It is built using Python and the discord.py library and is open for members of the society to contribute to the development of the bot. It announces events such as new members joining the server or can calculate simple math functions and allow users to create poll questions.
