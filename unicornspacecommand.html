Unicorn Space Command Game Jam Entry

Project Description

In 2013 I entered and won the UWS Windows 8 game jam, sponsored by Microsoft. I created Unicorn Space Command, a simple asteroids-like game where you need to shoot incoming asteroids that would damage your ship. At the end of the jam I had the game published on the Windows Store and had an interview published on various Microsoft websites like MSDN and uBelly. I also had the game featured on Scottishgames.net. A while after the jam I made some improvements to the game and published it on Google Play for Android devices. To date the game has had around 3000 downloads.
