Burn Zombie Burn Professional Credit

Project Description

Burn Zombie Burn was my main responsibility as a junior programmer intern at Tick Tock Games. It was previously published on a number of platforms and I was tasked with preparing the game to be published on iOS. I worked on bug fixing and modifying the game to run on a mobile device which included optimising the UI. I also added iOS Game Center integration for leaderboards and achievements.
