Real Time 3D Graphics Coursework

Project Description

This coursework was created for my first class dealing with 3D graphics. It allows you to control a character that can move around a lit and textured scene. You can switch to another character to move around to demonstrate multiple characters. It also has basic collision detection and resolution, keeping you within the bounds of the scene.
