TMX Tilemap Loader Open Source Project

Project Description

I created a simple C++ framework to load .tmx Tilemaps created in Tiled, a 2D tilemap editor. I created this to try out with some game engine development. It uses RapidXML to parse the .tmx files and I created a custom data structure to easily access the data for someone to easily adapt it to their own rendering and level storage design. This tool is now added to the list of helpful frameworks on the Tiled documentation site.
