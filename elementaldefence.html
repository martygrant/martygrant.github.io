Elemental Defence Group Project

Project Description

This was a group project for a generic project class. We had the freedom to develop a simple piece of software or game. We chose to create a simple tower defence game where you need to stop enemies destroying your base, by placing various turrets on the map. The rest of the group members handled design and planning and I implemented the actual game itself.
