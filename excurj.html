Excurj Group Project

Project Description

A web application aimed at connecting travellers and local residents willing to give friendly tours. Built for a group project. The site allows you to search for a cities and users. On a city page you can view a list of users, filter them and go to their individual pages where you can see their hobbies, languages and reviews by other users. You can also message a user or leave them a review if you have signed up and are logged in.
