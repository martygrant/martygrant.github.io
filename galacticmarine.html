Galactic Marine Coursework

Project Description

This coursework was created for my Games Engine Design class where we explored writing maintainable code using design patterns. The demo is an incredibly simple game but uses well defined patterns such as decorators, state management and factories. It was one of my best classes as it has completely changed how I approach writing game code, I understand now that using patterns is a must have.
