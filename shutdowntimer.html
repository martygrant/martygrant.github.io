Shutdown Timer Hobby Project

Project Description

I often used the Windows command prompt "shutdown" command to schedule a shutdown when downloading a big game or running a back up so I could leave the computer and have it turn off after a few hours. I decided to throw together a quick program to use the command with a GUI. I created the program using C# with Winforms. I plan to later use a cross-platform GUI so I can build a macOS and Linux version as I use all three operating systems.
