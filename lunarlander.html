Lunar Lander Coursework

Project Description

This is a piece of coursework created for my 2D Graphics Programming class. It is a simple lunar lander game where you have limited fuel and must find a flat area to land on. It features a simple random terrain generation function that always guarantees two flat areas for the lander to use.
