Space Station Zeta Coursework

Project Description

For my Intro to Games Development class I created my first game using GameMaker. I tried to merge some game genres (top down, side scrolling and space shooter) into one. As the player you need to try to escape a space station, fighting generic enemies and then shoot down an enemy spaceship. It was a fun process creating this game as I had freedom to create what I wanted and had a lot of enjoyment working on my first game.
